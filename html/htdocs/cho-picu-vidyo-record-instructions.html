Start PICU Vidyo™ Recording Instructions

Basic instructions to start recording a Vidyo™ conference with the Reach Transport team from the PICU Command Center Windows 7 desktop computer.

  1. Right mouse click the Vidyo™ Desktop application icon in the Windows 7 taskbar to display the drop down menu.

  2. Select "Manage Account" from the drop down menu.

  3. Click "Home" from the Vidyo™ Portal web page navigation menu.

  4. Click "PICUTransport" from the "My Contacts" tab on the Vidyo™ Portal web page.

  5. Click the "Join Room" button on the Vidyo™ Portal web page to start the video conference.

  6. Click "Control Meeting" from the Vidyo™ Portal web page navigation menu.

  7. Click "PICUTransport" to select the Room to record.

  8. Click the record button icon to start recording.

  9. Select "STANDARD-QUALITY" from the pop-up menu.
